Spotify Top 200

An interactive dashboard using global data from SpotifyCharts and Spotify API.
Click on a song below to see the number of daily streams or ranking in the Top 200.
If a song was not in the Top 200 a certain day, then no data is shown for that day.
Data is only available for the Top 200 songs on Spotify from 1/1/2018 - 5/28/2018, hence days in Top 200 and highest consecutive days are only considering this time period.
The graphs for a song only show data from the release date (or 1/1/2018, whichever is later) to the last date it was in the Top 200 (or 5/28/2018, whichever is earlier).
Use the dropdown menu to see the Top 200 chart for a different day in 2018. Streams refers to number of streams for that day.
Use the searches to find all songs containing a certain keyword/phrase, or search for an artist to see the songs that have reached the Top 200 in 2018.
(For song names/artists containing an accent, the search works best by searching the part up to the accent. Example: Beyoncé → Beyonc)
